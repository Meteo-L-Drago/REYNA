## Schritte im Supabase-Dashboard

1. Öffne **[supabase.com/dashboard](https://supabase.com/dashboard)** und wähle dein REYNA-Projekt.

2. Links auf **„SQL Editor“** klicken.

3. Auf **„+ New query“** klicken.

4. Den kompletten folgenden SQL-Code einfügen:

Siehe SQL unten – aus `supabase/migrations/002_restaurants.sql` kopieren oder direkt im Projekt öffnen.

5. Auf **„Run“** (oder Cmd+Enter) klicken.

6. Prüfen: unten sollte **„Success. No rows returned“** stehen.

7. Zum Testen: Links auf **„Table Editor“** → Tabelle **`restaurants`** sollte sichtbar sein.
